我是Obsidian的重度用户，我经常编写笔记，记录日常开发，也经常用Obsidian Web Clipper插件将网页保存为笔记。虽然可以使用Cursor直接打开Obsidian工作区进行提问，但毕竟有点怪怪的，所以平时还是编程时才打开Cursor。最近DeepSeek模型火出圈，所以想着基于DeepSeek+Obsidian来搭建个人知识库。

## 1、获取DeepSeek大模型的API密钥

本文写于DeepSeek官方API访问还没有恢复稳定，正好硅基流动推出了自托管的DeepSeek的V3模型，而且还在搞活动，注册就送14元，邀请用户双方都送2000万tokens，所以可以转用硅基流动，下面是我的邀请链接。

[https://cloud.siliconflow.cn/i/Czl0sFzp](https://cloud.siliconflow.cn/i/Czl0sFzp?ref=mksaas.me)

账号注册后，新建API密钥，通过这个密钥，我们就可以调用大模型了。

<img src="https://cdn.mksaas.me/images/dae27a6a3bbf57ea7e60ac0dfad0b232.png" alt="create api key in siliconflow" />

现在DeepSeek官方服务已经恢复，你也可以不用注册硅基流动，直接注册DeepSeek官方，然后新建一个API密钥。

<img src="https://cdn.mksaas.me/images/d5f684a9631ea6aa0a29398c8e41b836.png" alt="create api key in deepseek" />

## 2、在Obsidian的插件市场中搜索并安装插件Copilot

插件Copilot是一个支持以聊天框的形式对Obsidian工作区进行问答的插件。

<img src="https://cdn.mksaas.me/images/40d37bcbc4248cf40812ccf667a47584.png" alt="obsidian Copilot plugin" />

## 3、添加 `DeepSeek-V3` 模型到Copilot的Chat Models

<img src="https://cdn.mksaas.me/images/c109a5496b0122651d7df4f927b8856f.png" alt="add custom Chat Model" />

按照下面的参数进行配置，Model选择 `deepseek-chat` ，URL设置为 [`https://api.deepseek.com`](https://api.siliconflow.cn/v1?ref=mksaas.me) ，API Key就是上面创建的密钥。

<img src="https://cdn.mksaas.me/images/e0b3dfd9270dc1025caee5a3ff73f23b.png" alt="add DeepSeek-V3 model" />

如果你是选择硅基流动上的DeepSeek模型，则按照下面的参数进行配置，Model选择 `deepseek-ai/DeepSeek-V3` ，URL设置为 [`https://api.siliconflow.cn/v1`](https://api.siliconflow.cn/v1?ref=mksaas.me) ，API Key就是上面创建的密钥。

<img src="https://cdn.mksaas.me/images/434061384ec628c7e12bd56d7109a786.png" alt="add DeepSeek-V3 model" />

## 4、添加 `BAAI/bge-m3` 模型到Copilot的Embedding Models

这个模型是免费的，配置方式跟上面的硅基流动的DeepSeek模型配置方式类似，只是模型名称换一下，其他参数保持不变，API密钥就用硅基流动那个密钥。

<img src="https://cdn.mksaas.me/images/76128a64bc7e8f9dedae5724f20d430d.png" alt="add custom Embedding model" />

<img src="https://cdn.mksaas.me/images/a204785328b7732c07f9d254bb197ce2.png" alt="add BAAI/bge-m3 model" />

## 5、启用并设置默认的Chat Model和Embedding Model

<img src="https://cdn.mksaas.me/images/9d9b8f1cd447ab60336ad19d91cad390.png" alt="enable chat model" />

<img src="https://cdn.mksaas.me/images/e58094df08255b48d3b97d35c12eeb38.png" alt="enable embedding model" />

启用之后，在Basic选项卡中设置这2个新增模型为默认的模型。

<img src="https://cdn.mksaas.me/images/0f27dc5377fe32e1bd88045afd5c8eab.png" alt="set default models" />

## 6、测试对Obsidian知识库进行问答

点击Obsidian侧边栏新增的聊天按钮就可以打开聊天框，然后设置模式为vault QA，模型是DeepSeek-V3，然后就可以针对知识库进行提问了！

<img src="https://cdn.mksaas.me/images/d040cfc56e77554720675bb5ceea2d92.png" alt="chat box of Copilot" />

我让它帮我“梳理Mkdirs项目的时间线”，没想到它的结果很准确，Mkdirs确实是2024年10月21日发布，然后它列出了我什么时间点在什么reddit社区发了什么帖子，确实我有专门记录下发帖的情况，有这样的助理，是不是很神奇？

<img src="https://cdn.mksaas.me/images/68a0e3c7f629c95e66016df06a3df3c4.png" alt="qa for mkdirs p1" />

<img src="https://cdn.mksaas.me/images/0006008cca2ff337991ce286ca1478dc.png" alt="qa for mkdirs p2" />

又比如，我让它计算我去年在购买软件和服务上花了多少钱，它计算得确实不错。

<img src="https://cdn.mksaas.me/images/a46538b86b8bf2363c00546cb0e16d35.png" alt="calculate the money spent on softwares" />
