## 项目简介

这是一个创新的 AI 图像生成应用，利用最新的扩散模型技术，能够根据用户输入的文本描述生成高质量、高分辨率的图像。

## 核心功能

- **文本到图像生成**：输入文字描述，AI 自动生成对应图像
- **风格迁移**：支持多种艺术风格的图像转换
- **高分辨率输出**：支持最高 4K 分辨率的图像生成
- **批量处理**：可同时处理多个生成任务

## 技术架构

### 前端技术栈
- React 18
- TypeScript
- TailwindCSS

### 后端技术栈
- Python 3.10
- PyTorch 2.0
- FastAPI

### AI 模型
- Stable Diffusion XL
- ControlNet
- LoRA 微调
