## 项目介绍

1. 自我进化的智能体：AI 创造并积累工具
Xyzen 的核心是自演化。它不是一个只能使用预设工具的普通 Agent，而是能在与你的对话中，根据指令即时编写、部署并永久保存新工具。这意味着 Xyzen 会随着使用不断“成长”，逐步形成专属于你或你团队的强大工具库，实现能力的复利式增长，最终从一个通用助手进化为无所不能的专家伙伴。

2. 轻量级嵌入集成：一行命令为现有网站注入 AI
为现有网站集成 AI 对话从未如此简单。传统方案往往需要整站部署，或提供的 UI 组件简陋粗糙，迫使开发者从头造轮子。Xyzen 将强大、美观且高度可定制的对话能力打包成一个即插即用的 NPM 包 (@sciol/xyzen)。开发者只需一行 yarn add 命令，就能为任何现有 React 应用轻松集成一个生产级别的聊天机器人，省去你的大量开发时间。人生苦短，我用 LLM！

3. 绝对安全与隔离：为每个工具打造的容器沙箱
安全是 Xyzen 的基石。不同于在主进程中执行代码的方案，Xyzen 独创性地为每一次工具的执行都启动一个独立的 Docker 容器。这种“阅后即焚”的沙箱机制提供了极致的安全隔离，彻底杜绝了恶意代码影响宿主环境的风险，同时还能自动管理每个工具的复杂依赖，确保了系统的稳定与安全。
